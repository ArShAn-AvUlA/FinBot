# FinBot

FinBot is an AI-powered financial assistant that makes understanding stocks simple and conversational. Users can ask natural-language questions about stocks and receive clear insights like sector information, P/E ratios, market capitalization, dividend yields, and recent news headlines.

## Features
- Answer stock-related questions in plain language
- Compare company P/E to sector averages
- Market cap and dividend information
- Real-time news headlines from Yahoo Finance

## How to Run
1. Clone the repo: `git clone https://github.com/yourusername/FinBot.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run `finbot.py` or open `main.ipynb` in Google Colab
4. Ask FinBot your stock questions (e.g., "Should I buy AAPL?", "News about NVDA")

## Tech Stack
- Python
- yFinance, Pandas, NumPy
- BeautifulSoup + Requests (for live news)
- Google Colab compatible
